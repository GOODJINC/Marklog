## [1.2.0] - 2025-01-19

### Added
- **다국어 지원 (i18n)**: 한국어/영어 자동 전환
  - 브라우저 언어 자동 감지 (한국어 → 한국어, 그 외 → 영어)
  - 설정에서 수동 언어 변경 가능 (자동/English/한국어)
  - 모든 UI 텍스트, 버튼, 메시지 번역 지원
- **폴더 스크롤 모드**: 설정에서 폴더 높이 동작 방식 선택 가능
  - 고정 높이 (폴더 내부 스크롤): 페이지 스크롤 없이 폴더 내부만 스크롤
  - 자동 높이: 페이지 스크롤을 허용하며 폴더가 필요한 만큼 확장

### Changed
- **URL 입력 개선**: `https://` 프로토콜이 없어도 자동으로 추가하여 입력 간소화 (예: `naver.com` → `https://naver.com`)
- **폴더 높이 최적화**:
  - 2줄 모드: 280px 고정 높이
  - 1줄 모드: 480px 고정 높이 (더 넓은 화면 활용)
- **즐겨찾기 영역**: 80% 너비로 중앙 정렬하여 더 나은 레이아웃 제공
- **설정 패널 UI**: 중앙 모달에서 우측 슬라이드 패널로 변경하여 직관적인 UX 제공

### Fixed
- **페이지네이션 버튼**: 추가 버튼이 다음 페이지에 있을 때 페이지 이동이 작동하지 않던 문제 해결
- **즐겨찾기 높이 정렬**: 즐겨찾기 아이템과 추가 버튼의 높이를 동일하게 맞춤
- **키보드 네비게이션**: 첫 사용 후 동작이 멈추던 문제 수정
- **카드 불투명도**: 설정의 카드 불투명도가 실제로 적용되지 않던 문제 해결

## [1.1.0] - 2025-01-17

### Added

#### 키보드 네비게이션
- **방향키 이동**: 즐겨찾기, 폴더, 사이트 간 `↑↓←→` 키로 자유롭게 이동
- **Enter 열기**: 선택된 항목에서 Enter 키로 사이트 바로 열기
- **키보드 힌트**: 네비게이션 활성화 시 우측 하단에 단축키 안내 표시
- **포커스 하이라이트**: 현재 선택된 항목에 시각적 강조 표시

#### 폴더 드래그앤드롭
- **폴더 순서 변경**: 폴더 헤더를 드래그하여 원하는 위치로 순서 변경
- **2차원 그리드 지원**: 4열 그리드 레이아웃에서 자연스러운 드래그앤드롭

#### 배경 커스터마이징
- **단색 배경**: 원하는 색상으로 배경 설정
- **그라데이션 배경**: CSS 그라데이션 직접 입력 또는 프리셋 선택
  - 5가지 프리셋: 보라, 핑크, 파랑, 민트, 선셋
- **이미지 배경**: URL로 배경 이미지 설정
- **카드 불투명도**: 배경이 비치도록 카드 투명도 조절 (30%~100%)

#### 폴더 색상
- **개별 폴더 색상**: 각 폴더 헤더에 고유 색상 지정 가능
- **색상 선택기**: 컬러 피커와 HEX 코드 입력 모두 지원
- **초기화 버튼**: 기본 색상으로 쉽게 복원

### Changed
- 데이터 저장 키를 `linklogData`에서 `marklogData`로 변경 (기존 데이터 자동 마이그레이션)
- 설정 패널 UI 개선 (배경 설정 섹션 추가)

### Fixed
- 드래그앤드롭 시 간헐적으로 발생하던 요소 중복 문제 해결

## [1.0.0] - 2025-01-16

### Added

#### 핵심 기능
- **새 탭 대체**: 브라우저 새 탭을 Marklog 대시보드로 대체
- **즐겨찾기 영역**: 자주 사용하는 사이트 8개씩 페이지별 관리
- **폴더 기반 북마크**: 폴더별로 사이트 분류 및 관리 (4열 × 2행 그리드)
- **사이트 관리**: 각 폴더 내 사이트 추가/수정/삭제

#### 검색 기능
- **통합 검색**: 사이트 이름, URL, 메모에서 검색
- **초성 검색**: 한글 초성으로 검색 가능 (예: "ㄴㅇㅂ" → "네이버")
- **키보드 네비게이션**: 화살표 키로 검색 결과 이동, Enter로 바로 접속

#### UI/UX
- **검색창 가운데 정렬**: 상단 중앙에 검색창 배치
- **설정 버튼**: 우측 상단 설정 접근
- **파비콘 자동 표시**: Google Favicon API 활용
- **폴더 이모지**: 각 폴더에 이모지 아이콘 설정 가능
- **반응형 디자인**: 다양한 화면 크기 지원

#### 페이지네이션
- **즐겨찾기**: 8개 초과 시 페이지 전환 (좌우 화살표)
- **폴더**: 8개 초과 시 페이지 전환 (좌우 화살표)
- **페이지 인디케이터**: 현재 페이지 / 전체 페이지 표시

#### 설정
- **테마**: 라이트/다크 모드 전환
- **즐겨찾기 표시**: 즐겨찾기 영역 숨기기/표시 설정
- **폴더 줄 수**: 1줄 또는 2줄 선택
- **강제 동기화**: 수동 동기화 버튼
- **데이터 백업/복원**: JSON 파일로 내보내기/가져오기

#### 편의 기능
- **클릭 옵션**: 
  - 일반 클릭: 현재 탭에서 열기
  - Ctrl+클릭: 새 탭에서 열기
  - Shift+클릭: 새 창에서 열기
- **키보드 단축키**:
  - `/`: 검색창으로 이동
  - `Esc`: 모달/검색 닫기
  - `↑↓`: 검색 결과 이동
  - `Enter`: 선택한 사이트 열기
- **드래그앤드롭**: 즐겨찾기, 사이트 순서 변경 및 폴더 간 이동
- **길게 누르기**: 즐겨찾기 항목 수정 모달 열기

#### 데이터 관리
- **Chrome Storage Sync**: 구글/마이크로소프트 계정 동기화 지원
- **LocalStorage 폴백**: 개발/테스트 환경 지원
- **샘플 데이터**: 첫 설치 시 예시 데이터 제공

### Technical Details
- **Manifest Version**: 3 (최신 Chrome Extension 표준)
- **지원 브라우저**: Chrome, Edge (Chromium 기반)
- **외부 의존성**: 없음 (순수 HTML/CSS/JavaScript)
